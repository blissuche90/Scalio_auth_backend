# Scalio_auth_backend

# auth Sample using Nest 


# To Use:
Requires: node and mongoDB

1. clone this repository
2. cd Scalio_auth_backend


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```


# ENDPOINTS:


1. /login [POST]     [For Login purposes]
2. /signup [POST]    [Used For Registration Purposes]
3. /profile [POST]    [Retrieve user details]
